# GAR PRD

## 1. 제품 요약

GAR은 나라장터(G2B)의 차량 렌트/임차 입찰공고를 자동 수집하고, 첨부문서를 구조화해, 렌트사별로 맞춤 알림과 견적 생성을 제공하는 SaaS다.

## 2. 해결하려는 문제

- 중소 렌트사는 나라장터를 수시로 모니터링할 전담 인력이 없다.
- 공고 제목만으로는 실제 렌트 기회인지 구분하기 어렵다.
- 공고마다 HWP/PDF 조건을 사람이 읽어야 해서 속도가 느리다.
- 자격요건과 적격심사 기준을 빠르게 판단하기 어렵다.
- 견적 산정이 매번 수기와 경험치에 의존한다.

## 3. 타겟 사용자

### 1차 고객

- 자동차대여사업자 (업종코드 1457)
- 5~50대 규모의 중소 렌트사
- 영업 담당자 1~3명이 입찰 탐색, 검토, 견적까지 맡는 조직

### 2차 고객

- 다지점 운영 렌트사 영업팀
- 입찰 담당과 견적 담당이 분리된 중대형 사업자

## 4. 핵심 가치 제안

- 렌트 공고를 자동으로 모아준다.
- "이 공고가 뭘 원하는지" AI가 구조화해서 보여준다.
- 내 소재지·프로필 기준으로 "나 자격 되냐" 바로 알려준다.
- 적격심사 시뮬레이션으로 "입찰하면 붙을 수 있나" 점수 예측.

## 5. 핵심 사용자 시나리오

1. 사용자는 대시보드에서 오늘 새로 올라온 공고와 마감 임박 공고를 본다.
2. 공고 목록에서 지역, 차종, 예산, 마감일로 필터링한다.
3. 상세 화면에서 AI 분석 결과와 원문 첨부파일을 함께 확인한다.
4. 회사 프로필과 비교한 자격/매칭 결과를 본다.
5. 적합한 공고는 알림을 받고, 적격심사 시뮬레이션으로 가능성을 확인한다.

## 6. MVP 범위

### 포함

- 나라장터 수집기
- 차량 렌트 타겟 필터링
- 공고 목록/상세
- HWP/HWPX/PDF 다운로드 및 파싱
- AI 분석 카드 ("이 공고가 뭘 원하는지")
- 자격 체크 (소재지 기반 1차, 프로필 기반 상세)
- 적격심사 시뮬레이션 (프로필 기반 점수 예측)
- 텔레그램/이메일 알림
- 회사 프로필 (소재지 필수, 점진적 온보딩)

### 제외

- 견적서 자동 생성
- 단가표 관리
- 결제/구독 자동 청구
- OCR 기반 스캔 PDF 고정밀 복원
- 모바일 앱
- 외부 API 제공

## 7. 요금제 가설

| 플랜 | 월 요금 | 핵심 제한 |
|------|---------|----------|
| Free | 0원 | 3일 지연, 일일 5건 열람 |
| Basic | 49,000원 | 실시간 알림, 기본 필터 |
| Pro | 99,000원 | AI 분석 카드, 적격심사 시뮬레이션, 매칭 알림 |

## 8. 성공 지표

- 주간 활성 고객사 수
- 고객사당 저장/열람 공고 수
- 수집 공고 중 유효 렌트 공고 비율
- AI 분석 성공률
- 알림 클릭률
- 시뮬레이션 사용율
- 유료 전환율

## 9. 주요 가정

- 차량 렌트 공고는 월 100건 이상 지속적으로 발생한다.
- 제목/분류코드/첨부 분석을 결합하면 유효 공고를 충분히 좁힐 수 있다.
- 렌트사는 완성형 ERP보다 "공고 발견 + 해석 + 자격 확인"에 먼저 비용을 지불한다.

## 10. 주요 리스크

- G2B 응답 구조와 첨부파일 다운로드 규칙 변경
- HWP/HWPX 파싱 품질 편차
- 스캔본 PDF 비중 증가
- 공고마다 적격심사 기준표가 달라 단일 산식이 깨질 가능성
